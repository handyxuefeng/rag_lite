# 1. RAG Lite 
- 一个轻量级、可扩展的 RAG（检索增强生成）系统，使用 LangChain 和 DeepSeek API 实现。支持多种后端存储、数据库和向量数据库，易于扩展和部署。
# 2. 核心特性
- 知识库管理: 支持创建、查询、更新、删除知识库，支持多知识库隔离
- 文档处理: 支持 PDF、DOCX、TXT、MD 格式文档上传和解析，自动分块和向量化
- 智能检索: 支持向量检索、关键词检索、混合检索和重排序（Rerank）
- 智能问答: 基于 RAG 的问答系统，支持流式输出和引用溯源
- 用户系统: 完整的用户注册、登录、会话管理功能
- Web 界面: 美观的 Web 界面，支持知识库管理、文档管理和智能问答
- 系统设置: 可配置 LLM 模型、Embedding 模型、检索参数等
- 可扩展架构: 支持多种后端存储、数据库和向量数据库，易于扩展

# 3. 架构设计
存储服务
本地文件系统 (默认)
MinIO 对象存储 (可选)
3.1.2 数据库
MySQL (默认)
MongoDB (可选)
3.1.3 向量数据库
ChromaDB (默认)
Milvus (可选)
3.1.4 LLM 模型
DeepSeek (默认)
OpenAI
Ollama
支持自定义扩展
3.2 技术栈
后端框架: Flask 3.0+
数据库: MySQL 5.7+ / MongoDB
向量库: ChromaDB / Milvus
文件存储: 本地文件系统 / MinIO
LLM: DeepSeek API / OpenAI / Ollama
Embedding: 本地 HuggingFace 模型（sentence-transformers）
文档处理: LangChain 文档加载器和文本分割器
项目管理: UV（快速依赖管理）
日志系统: 统一的日志管理，支持文件轮转
